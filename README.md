# CodeAlpha AI Internship Projects

## Task 1: Language Translation Tool
A python application that translate text between multiple languages using the deep translator library. Features a simple GUI built with Tkinter.

## Task 2: FAQ Chatbot
A python chatbot that answers programming related queries using NLP techniques. Uses TF-IDF vectorization andc cosine similarity to match user question with the most 
relevent FAQ answer. Features a chat UI built with Tkinter.

##Libraries Used
- deep-translator
- scikit-learn
- nlkt
- tkinter

## How to RUN
1. Install the required libraries: 'pip install deep-translator scikit-learn ntkl'
2. Run 'translator.py' for the translation tool.
3. Run 'chatbot.py' for the FAQ chatbot.
